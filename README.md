# Titanic Survival Prediction 

##  Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques. It is built using Python and uses classification algorithms to analyze survival patterns.


##  Dataset
* Source: Kaggle Titanic Dataset
* File Used: Titanic-Dataset.csv
* Features include passenger class, gender, age, fare, and more
##  Technologies Used
* Python
* Pandas
* NumPy
* Seaborn & Matplotlib
* Scikit-learn
## Project Workflow
1. Data Loading and Exploration
2. Data Cleaning (handling missing values)
3. Feature Engineering (FamilySize, IsAlone)
4. Encoding categorical variables
5. Feature Scaling using StandardScaler
6. Model Training and Evaluation
##  Machine Learning Models Used
* Random Forest Classifier
* Logistic Regression
* Decision Tree Classifier
## Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix
* Cross Validation
## Key Features / Improvements

* Implemented feature engineering to improve model performance
* Applied feature scaling for better convergence
* Compared multiple ML models
* Visualized data using plots and heatmaps
* Analyzed feature importance
## Results
* Achieved good accuracy using Random Forest model
* Model performance improved after feature engineering and scaling
